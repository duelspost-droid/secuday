# NotebookLM 소스 팩 — 2026년 7월 정보보호의 날

> NotebookLM에 **이 문서를 텍스트 소스로 붙여넣고**, 아래 "추가 소스 URL"도 함께 넣은 뒤,
> ① 4컷 만화 스크립트/팩트체크를 받고 ② 한국어 오디오 오버뷰를 생성한다.
> 만화(SVG 4컷)는 secuday 템플릿이 렌더하고, 오디오는 NotebookLM이 담당한다(역할 분담).

## 주제
가족·지인 사칭 **메신저피싱**(카카오톡·문자) — "엄마 나 폰 고장났어" 수법

## 핵심 팩트 & 통계 (근거)
- 2025년 보이스·메신저피싱 **피해액 약 1조 1,330억 원**(전년 대비 +56%), **건수 21,588건**(+15.6%). — 금융감독원 2026.4
- 정부 **종합대책(2025.8)** 이후 **7개월 연속(2025.10~2026.4) 피해 감소**.
- 메신저피싱 수법: 가족·지인 사칭 → "폰이 고장났다/전화가 안 된다" → **신분증·상품권 핀번호·원격제어 앱** 요구 → 송금·정보 탈취.
- 핵심 방어: **메시지가 아니라 저장된 번호로 직접 전화**해 본인 확인. 회사 자금·결재 요청도 유선/대면 재확인.

## 4컷 만화 스크립트 (상황 → 함정 → 피해 → 수칙)
1. (평온) 사칭범(엄마): "아들~ 엄마 폰 고장나서 PC로 연락해. 급한데 좀 도와줄래?" / 캡션: 가족을 사칭한 메신저 한 통으로 시작된다
2. (불안) 사칭범(엄마): "이 링크로 신분증이랑 상품권 핀번호만 찍어 보내줘!" / 캡션: 신분증·상품권·링크를 요구하면 100% 사기
3. (충격) 직원: "…보냈는데 엄마가 '그런 적 없다'네?" / 캡션: 확인 없이 보내면 돈도 정보도 한순간에
4. (안도) 직원: "메시지 말고 엄마 번호로 직접 전화해 확인했어요!" / 캡션: 오늘의 수칙 — 끊고, 본인에게, 직접 전화

## 임직원 보안 수칙 (오늘의 보안수칙)
1. 가족·지인의 '송금/상품권' 요청은 끊고 본인에게 직접 전화로 확인한다
2. 신분증·계좌·비밀번호 사진을 메신저로 보내지 않는다
3. 출처 불명 링크·앱(원격제어) 설치 요구는 즉시 거절한다
4. '다른 번호로 연락' '전화가 안 된다'는 말은 의심 신호로 본다
5. 피해가 의심되면 즉시 112·금융사·정보보호팀에 신고한다

## NotebookLM 작업 순서
1. 새 노트북 생성 → 위 소스 URL 추가 + 이 문서를 '붙여넣은 텍스트' 소스로 추가.
2. 채팅에 요청: "위 소스를 근거로 ① 핵심 사실·수치 팩트체크 ② 임직원 대상 4컷 만화 스크립트(상황→함정→피해→수칙) ③ 5개 보안 수칙을 한국어로 정리해줘."
3. **Audio Overview(오디오 개요)** 생성 → 언어 **한국어**, 커스텀 지시:
   "금융회사 임직원 대상 보안 인식 브리핑. 가족 사칭 메신저피싱 예방 중심으로, 4컷 만화 흐름(상황→함정→피해→수칙)을 짧고 쉽게. 5분 내외, 마지막에 오늘의 보안수칙 5개 요약."
4. 생성된 오디오 다운로드(.wav/.mp3) → 만화 뉴스레터와 함께 배포(또는 사이트에 링크/플레이어 추가).
